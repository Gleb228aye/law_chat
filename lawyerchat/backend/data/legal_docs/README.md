# Юридические документы

Сюда нужно вручную положить `.txt` или `.jsonl` файлы с юридическими документами.

Форматы:

- `.txt` — простой режим, файл разбивается на чанки через `split_legal_text`.
- `.jsonl` — структурированный режим, каждая строка является отдельным готовым юридическим фрагментом.

Примеры имен файлов:

- `trudovoy_kodeks.txt`
- `grazhdanskiy_kodeks.txt`
- `example_legal_corpus.jsonl`

После добавления или обновления файлов запустите индексацию:

```bash
python -m scripts.ingest_documents
```

Или используйте endpoint:

```text
POST /api/documents/reindex
```

Не добавляйте сюда автоматически скачанные документы. База текстов для MVP наполняется вручную.
